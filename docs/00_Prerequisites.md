# Prerequisites

Before installing Armbian, the ZTE B860H Set-Top Box must be rooted.

Root access is required to:

- Gain full system access.
- Enable the necessary boot configuration.
- Prepare the device for installing Armbian.

Once the rooting process has been completed successfully, continue with the Armbian installation described in this document.

> **Note**
> The rooting procedure may vary depending on the firmware version installed on the ZTE B860H.
