# 🛰️ Low-Power Smart Home Gateway

### Android TV Box → Armbian Linux → Docker IoT Server

A complete low-power IoT gateway built by converting a standard Android TV Box (STB) into a headless Linux server capable of running Home Assistant, Node-RED, Mosquitto MQTT, and ZeroTier simultaneously.

Designed for 24/7 operation with very low power consumption while supporting both offline local automation and secure remote access.

# 📖 About

This project demonstrates how an inexpensive Android TV Box can replace a traditional mini PC or Raspberry Pi as the core of a smart home infrastructure.

Instead of running Android TV, the STB is converted into an Armbian Linux server and optimized for continuous operation.

The architecture focuses on:

- Low Power Consumption
- Docker Containerization
- Local Network Automation
- Secure Remote Access
- Offline First Design
- Easy Maintenance
- Modular Services

# 🎯 Project Goals

- Convert a low-cost Android STB into a reliable Linux server.
- Minimize power consumption.
- Run Home Assistant inside Docker.
- Deploy Node-RED automation.
- Provide MQTT messaging.
- Enable secure remote access.
- Operate continuously for 24 hours.
- Maintain local automation even without internet access.
